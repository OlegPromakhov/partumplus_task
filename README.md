## Usage

To execute tests locally please run following commands in terminal:

1. In headless mode

```sh
   npm run cy:run
   ```
   
2. In Chrome browser

```sh
   npm run cy:run:chrome
   ```
   
3. To open Cypress GUI

```sh
   npm run cy:open
   ```
   
   Enjoy!
